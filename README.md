## ✨ 功能

- **🫁 身體部位選擇** — 頭部、胸部、腹部、四肢等部位分類
- **✅ 症狀勾選** — 每個部位對應常見症狀清單
- **🏥 科室推薦** — 根據症狀組合智能推薦應掛科室（內科、外科、皮膚科等）
- **🚨 緊急提示** — 識別需立即就醫的危險症狀
- **💾 本地儲存** — 資料儲存在 localStorage

> ⚠️ **免責聲明**：本工具僅供參考，不能取代專業醫療診斷。如有嚴重症狀請立即就醫。
